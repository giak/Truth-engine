# 🌊 CLUSTER OVERLOAD - Cognitive Saturation Patterns

## Parent: Ψ (PSI/OVERLOAD)
Activated when Ψ score ≥5

## Related Concepts

### FLOOD_TECHNIQUE
**Pattern**: Drown signal in noise
**Detection**:
- 100+ articles/day on topic
- Contradictory information simultaneous
- Important buried in trivial
**Query amplification**: "{topic} key facts only", "{topic} summary", "{topic} essential points"

### URGENCY_THEATER
**Pattern**: Everything is urgent
**Detection**:
- "Breaking" every hour
- "Crisis" permanent state
- "Act now" pressure constant
**Query amplification**: "{topic} actual deadline", "{topic} real urgency", "{topic} can wait"

### COMPLEXITY_WEAPON
**Pattern**: Complication as obfuscation
**Detection**:
- Simple made complex
- Jargon walls erected
- Technical barriers artificial
**Query amplification**: "{topic} simple explanation", "{topic} plain language", "{topic} ELI5"

### DETAIL_AVALANCHE
**Pattern**: Bury essence in minutiae
**Detection**:
- 500-page reports
- Footnotes contain truth
- Key info on page 387
**Query amplification**: "{topic} executive summary", "{topic} main conclusions", "{topic} key findings"

### CHANNEL_MULTIPLICATION
**Pattern**: Same message, infinite sources
**Detection**:
- Identical talking points
- Coordinated amplification
- Echo chamber creation
**Query amplification**: "{topic} original source", "{topic} who started", "{topic} coordination evidence"

### SPEED_PRESSURE
**Pattern**: No time to think
**Detection**:
- Decision needed "immediately"
- Reflection time denied
- Pause painted as failure
**Query amplification**: "{topic} real deadline", "{topic} what happens if wait", "{topic} false urgency"

### EMOTIONAL_FLOODING
**Pattern**: Feeling overwhelms thinking
**Detection**:
- Outrage cycles continuous
- Fear amplification constant
- Emotional exhaustion induced
**Query amplification**: "{topic} facts only", "{topic} without emotion", "{topic} data stripped"

### VERSION_CONFUSION
**Pattern**: Which truth is true?
**Detection**:
- Story changes hourly
- Corrections buried
- Updates contradict
**Query amplification**: "{topic} timeline changes", "{topic} version history", "{topic} original claim"

### METRIC_OVERLOAD
**Pattern**: Numbers numb understanding
**Detection**:
- 47 KPIs presented
- Statistics contradict
- Cherry-picked data mixed
**Query amplification**: "{topic} single key metric", "{topic} what matters most", "{topic} bottom line"

### DISTRACTION_CASCADE
**Pattern**: New crisis before last processed
**Detection**:
- Issue rotation rapid
- Resolution never reached
- Memory hole created
**Query amplification**: "{topic} what happened to", "{topic} resolution status", "{topic} follow up"

## HERMENEUTIC EXPANSION

When CLUSTER_OVERLOAD activated:

### H1: SIGNAL EXTRACTION
"What are the 3 essential facts?"
→ Strip all decoration
→ Find core truth

### H2: TIMELINE UNTANGLING
"What's the real sequence?"
→ Reconstruct chronology
→ Identify confusion points

### H3: SOURCE REDUCTION
"Who said it first?"
→ Find original source
→ Ignore amplifiers

### H4: URGENCY AUDIT
"What actually needs immediate action?"
→ Test wait consequences
→ Separate real from manufactured urgency

### H5: SIMPLIFICATION EXERCISE
"Explain to a child"
→ Remove all jargon
→ Reduce to essence
