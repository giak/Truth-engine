## L'Herméneutique Tripartite : Révolutionner l'Analyse Critique

Traditionalement, l'analyse critique se contentait d'opposer "vrai" et "faux" selon une perspective unique. Truth Engine révolutionne cette approche en adoptant une **perspective dialectique** inspirée de la méthode académique la plus rigoureuse : l'herméneutique.

### Les Trois Axes d'Analyse

L'innovation fondamentale réside dans l'adoption simultanée de **trois perspectives complémentaires** qui révèlent les tensions dialectiques cachées dans les récits "consensuels" :

#### 1. L'Axe Académique ⟐🎓 : Les Sources Institutionnelles

Cette perspective examine le sujet à travers le prisme des **sources mainstream et institutionnelles** : médias traditionnels, think tanks établis, institutions académiques, rapports gouvernementaux.

_Exemple d'application_ : Pour analyser la "crise climatique", l'axe académique examine les rapports du GIEC, les déclarations des agences spatiales, les études des universités prestigious.

**Intérêt** : Ces sources offrent généralement une **base factuelle solide** et des méthodologies éprouvées. Elles constituent le "socle de référence" sur lequel construire l'analyse.

**Limites** : Ces sources sont souvent **financièrement et politiquement contraintes**, reflet des intérêts dominants dans la société.

#### 2. L'Axe Dissident 🔥⟐̅ : Les Voix Marginalisées

Cette perspective explore les **voix censurées, marginalisées ou qualifiées de "complotistes"** : médias alternatifs, chercheurs indépendants, activistes, sources qualifiées d'"adversaires" par les narrative mainstream.

_Exemple d'application_ : Pour la crise climatique, l'axe dissident examine les travaux de chercheurs indépendants sur les cycles solaires, les analyses géopolitiques sur la transition énergétique, les critiques du green capitalism.

**Intérêt** : Ces sources révèlent souvent des **angles morts intentionnels** et des intérêts cachés derrière les narratifs dominants.

**Risques** : Ces sources peuvent contenir des **désinformation intentionnelles** ou des biais ideologiques marqués.

#### 3. L'Arbitrage ◈◉○ : La Synthèse Basée sur les Preuves Primaires

Cette perspective finale ne se contente pas de "moyenner" les deux premières ; elle **arbitre les contradictions** en s'appuyant sur les **sources primaires** : documents officiels, données brutes, témoignages directs, archives.

_Exemple d'application_ : Pour la crise climatique, l'arbitrage examine les données températures brutes des stations météo, les rapports internes des compagnies pétrolières, les communications diplomatiques déclassifiées.

**Principe fondamental** : Les **sources primaires** (◈) ont préséance sur toutes les interprétations secondaires (◉○).

### La Révolution Dialectique

Cette approche révèle que la plupart des sujets "controversés" ne sont pas des débats entre "vrai" et "faux", mais des **tensions dialectiques** entre perspectives partiales.

_Exemple concret_ : La question du "chômage 7,4%" analysée dans l'introduction révèle cette tension dialectique :

- **Axe Académique** : "7,4% selon les critères BIT, amélioration significative"
- **Axe Dissident** : "Chiffres manipulés, réalité 2,33 fois plus élevée"
- **Arbitrage** : "Définition BIT = choix méthodologique, non manipulation. Mais silence sur halo et sous-emploi = omission intentionnelle"

L'arbitrage révèle que **les deux positions contiennent une part de vérité**, mais que l'omission du "halo chômage" constitue une **manipulation méthodologique** plutôt qu'une erreur factuelle.

## L'Epistemic Diversity Index (EDI) : Mesurer la Diversité des Sources

Au-delà des concepts de manipulation, Truth Engine intègre une **métrique révolutionnaire** : l'Epistemic Diversity Index (EDI), qui mesure la **diversité épistémique** des sources utilisées dans une investigation.

### Les Six Dimensions de l'EDI

L'EDI calcule la diversité selon **six dimensions complémentaires** :

#### 1. Diversité Géographique (25% du poids)

Mesure la **représentation géographique** des sources :

- Sources locales/régionales vs internationales
- Pays du Nord vs Pays du Sud
- Perspectives occidentales vs non-occidentales

#### 2. Diversité Linguistique (20% du poids)

Évalue la **pluralité linguistique** :

- Sources en langue originale vs traductions
- Langues majeures vs minoritaires
- Contextes culturels différents

#### 3. Diversité Stratégique (20% du poids)

Analyse la **position des sources** :

- Mainstream vs alternatifs
- Institutionnels vs indépendants
- Pro-establishment vs critiques

#### 4. Diversité de Propriété (15% du poids)

Examine qui **contrôle les sources** :

- Médias privés vs publics
- Groupes industriels vs associatifs
- Concentration vs pluralité

#### 5. Diversité Perspective (15% du poids)

Mesure la **pluralité des points de vue** :

- Perspectives académiques vs activistes
- Consensus vs dissidence
- Orthodoxie vs hétérodoxie

#### 6. Diversité Temporelle (5% du poids)

Évalue la **perspective temporelle** :

- Sources contemporaines vs historiques
- Urgence vs long terme
- Immédiateté vs contextualisation

### L'Application Pratique : L'Exemple du Chômage

Reprenons l'exemple du chômage 7,4% pour illustrer le calcul de l'EDI :

**Sources utilisées** :

- **Académique** : INSEE (gouvernemental français), Eurostat (européen)
- **Dissident** : Alternatives Économiques, Politis
- **Primaires** : Données BIT originales, rapports méthodologiques

**Calcul EDI** :

- **Géographique** : 0,3 (France + UE, manque perspectives non-occidentales)
- **Linguistique** : 0,2 (français principalement, peu d'international)
- **Stratégique** : 0,7 (mix institutionnel/alternatif)
- **Propriété** : 0,4 (publics + associatifs, manque privés)
- **Perspective** : 0,6 (académique + critique)
- **Temporelle** : 0,8 (contemporain + méthodologique)

**EDI Total** : (0,3×0,25) + (0,2×0,20) + (0,7×0,20) + (0,4×0,15) + (0,6×0,15) + (0,8×0,05) = **0,43/10**

**Classification** : Investigation de **qualité moyenne** (target ≥0,50 pour MEDIUM). Amélioration possible en ajoutant sources internationales et perspectives non-occidentales.

### Les Targets d'EDI par Complexité

Truth Engine définit des **seuils d'EDI** selon la complexité du sujet :

- **SIMPLE** : EDI ≥ 3,0/10
- **MEDIUM** : EDI ≥ 5,0/10
- **COMPLEX** : EDI ≥ 7,0/10
- **APEX** : EDI ≥ 8,0/10

Ces targets garantissent que chaque investigation bénéficie d'une **diversité épistémique minimale** adaptée à sa complexité.
