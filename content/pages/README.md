# Content Pages

Ce dossier contient les contenus statiques du Substack et autres plateformes.

## Structure

```
content/pages/
├── about.md          # Page "À propos" du Substack
├── welcome.md        # Page d'accueil (optionnel)
└── README.md         # Ce fichier
```

## Types de contenu produits

### 1. Tweets / Réponses rapides
- Format court (280 caractères) à moyen
- Multi-versions : courte / longue / pamphlet
- Style : percutant, ironique, direct
- Localisation : `outputs/tweets/` et `outputs/results/tweet-*.md`

### 2. Articles de fond (Substack)
- Format long (2000-5000 mots)
- Méthodologie Truth Engine : chronologies, sources primaires, preuves
- Concepts : "Modèle Mercure", "Signal d'Attaque", "Dogs of War"
- Localisation : `outputs/articles/`

### 3. Investigations
- Documents de travail, analyses forensiques
- Localisation : `investigations/` et `outputs/investigations/`

## Thématiques

- **Censure & contrôle de l'information** : Viginum, DSA, Mediapart, fact-checking
- **Affaire Epstein** : Réseaux d'élites, impunité, compromission
- **Politique française** : Macron, agriculture, budget, institutions
- **Géopolitique** : Iran, Russie, UE

## Processus de publication

1. Investigation → `investigations/`
2. Synthèse article → `outputs/articles/`
3. Adaptation tweets → `outputs/tweets/`
4. Publication Substack (manuelle)
5. Archivage contenus pages ici pour traçabilité
