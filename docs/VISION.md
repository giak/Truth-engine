## L'État Actuel — La Boucle

**Truth Engine** est un système cognitif qui apprend.

```
Investigation → Métriques → Feedback → Amélioration → Investigation
```

Chaque investigation génère des métriques. Les métriques révèlent les failles. Les failles sont corrigées. Le système grandit.

### Les Métriques

| Métrique | Description |
|----------|-------------|
| **EDI** | Indice de diversité épistémique (0-1). Plus il est haut, plus les sources sont diverses et indépendantes. |
| **ISN** | Indice de synthèse d'investigation. Mesure la qualité de la synthèse. |
| **Wolves** | Prédateurs détectés — acteurs qui profitent de la manipulation. |
| **H7** | Couverture sources adverses — les voix contraires sont représentées. |

### L'Amélioration Continue

Quand EDI est bas: les sources manquent de diversité → améliorer les templates de recherche.
Quand Wolves est bas: les bénéfices ne sont pas Named → améliorer le détection.
Quand H7 est bas: les voix contraires sont absentes → ajouter des sources.

Nous ne prétendons pas à la perfection. Nous prétendons à la progression.
