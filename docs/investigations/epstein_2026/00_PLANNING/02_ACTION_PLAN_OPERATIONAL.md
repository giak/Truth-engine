- Mitigation : Structure rigoureuse fichiers + synthèses quotidiennes

- Mitigation : Sources ◈ prioritaires, label speculation clairement

**R3 : Burnout (sujet éprouvant)**
- Mitigation : Pauses régulières, rotation domaines

- Mitigation : Only claim what can be proven, hedge when necessary

- Mitigation : Sauvegarder TOUT localement, backups multiples

**Ne JAMAIS :**
- Présenter hypothèse comme fait

- Négliger sauvegarde données
