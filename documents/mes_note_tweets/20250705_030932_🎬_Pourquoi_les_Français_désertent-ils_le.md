🎬 Pourquoi les Français désertent-ils les salles de cinéma ?
Ce n’est pas qu'une affaire de prix.
C’est une rupture culturelle.
Le cinéma, autrefois art populaire et subversif, s’est figé dans une forme de conformisme subventionné.
Il ne raconte plus le monde : il le filtre, le reformate.
Les œuvres se ressemblent, les récits s'autocensurent, les sujets dérangent peu, sauf le réel.
Ce désamour n'est pas passif : c’est une forme de résistance douce.
Les spectateurs, par leur absence, expriment un rejet du cinéma devenu outil de normalisation idéologique.
Le boycott est silencieux, mais profond.
La lumière s’éteint. Le message, lui, est clair.
