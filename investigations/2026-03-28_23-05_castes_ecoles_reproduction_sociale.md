## §0 THÈSE

> Les grandes écoles et universités d'élite ne sont pas des ascenseurs sociaux. Ce sont des **machines à reproduction de caste** qui transforment le capital hérité en capital méritocratique, créant des réseaux de cooptation qui structurent le pouvoir politique et économique.

## §1 BOURDIEU — LA NOBLESSE D'ÉTAT (1989)

### Concepts clés

**Capital culturel hérité** : connaissances, goût, aisance sociale transmis par la famille. Les enfants de cadres héritent d'un « code culturel » qui correspond exactement à ce que les grandes écoles testent.

**Homologie structurale** : la hiérarchie entre grandes et petites écoles reproduit la hiérarchie entre classes sociales. L'ENA en haut, la fac en bas = la bourgeoisie en haut, les classes populaires en bas.

**L'école comme légitimation** : l'école ne crée pas les inégalités, elle les **transforme en mérite**. L'héritage culturel devient « talent » et « travail ». Le fils de prof à l'ENA est « méritant » ; l'ouvrier qui n'y arrive pas est « incompétent ».

### Formule de Bourdieu
```
Capital scolaire = f(capital culturel hérité) + f(travail scolaire)
=> Plus le capital hérité est élevé, plus le travail scolaire est « efficace »
=> L'école reproduit la structure sociale en la légitimant
```

- 15 seniors sélectionnés chaque année par les membres existants
- Initiation rituelle : nuit dans la « Tombe » (bâtiment sans fenêtres), aveux personnels, squelette, serment de secret

## §4 ANALYSE DE CASTE

### Les 4 mécanismes de reproduction

1. **Sélection par le capital culturel** : les concours testent un « code » que seuls les héritiers maîtrisent
2. **Cooptation par le réseau** : les anciens d'une école embauchent les nouveaux (homophilie)
3. **Endogamie sociale** : les élites se marient entre elles (homogamie)
4. **Fermeture informationnelle** : les décisions se prennent dans des cercles fermés (Le Siècle, Bilderberg)

### Indice de fermeture élitaire (formule)

```
IF = (capital_culturel_hérité × 0.3) + (réseau_cooptation × 0.3) + (endogamie × 0.2) + (opacité_cercles × 0.2)
IF > 0.7 → CASTE FERMÉE
IF 0.4-0.7 → CLASSE DOMINANTE OUVERTE
IF < 0.4 → MÉRITOCRATIE RÉELLE
```

Estimation France 2026 : **IF ≈ 0.75** → caste fermée avec vernis méritocratique
