### Chaîne 1: NATURE → MORT
Enfants DRC → cobalt → batteries → data centers IA → profits Glencore/CMOC → actionnaires AXA/fonds pension

### Chaîne 2: TRAVAIL → PROFIT
Kényans 2$/h → nettoyage ChatGPT → IA "innovante" → profits OpenAI → actionnaires Microsoft → fonds pension

### Chaîne 3: RENTE → CONTRÔLE
AXA/Allianz → propriété locative → loyers élevés → classes moyennes bloquées → incapacité d'acheter → dépendance locative → profits assurances

### Chaîne 4: GUERRE → EXTRACTION
Thales 22Mds€ → armes → conflits → extraction ressources → profits → actionnaires → fonds pension

### Chaîne 5: NARRATIF → PARALYSIE
Brivael (croissance) / UE (décroissance) → débat → paralysie → personne ne voit le moteur → moteur continue

### Chaîne 6: DETTE → DOMINATION
