# ENQUÊTE 2 — Le climat comme variable de rupture : la crise qui pourrait forcer la réforme ou accélérer l'effondrement

**Date:** 2026-04-01 | **Complexité:** APEX | **Zone grise:** TROU MAJEUR

---

## §0 RÉSUMÉ

Le Climate Inequality Report 2025 (World Inequality Lab) révèle un fait qui relie directement la concentration de richesse à la crise climatique : le top 1% cause 41% des émissions mondiales via la PROPRIÉTÉ de capital privé. Pas seulement via la consommation (15%) mais via la propriété des entreprises qui émettent. La concentration de richesse EST la cause de la crise climatique.

Deux scénarios possibles :
1. Le climat FORCE la réforme : les coûts physiques dépassent les profits → les élites intègrent le risque → taxation carbone → redistribution forcée
2. Le climat ACCÉLÈRE la concentration : les riches achètent les terres habitables, les « stranded assets » détruisent les petits porteurs, l'adaptation est privatisée

Lequel l'emporte ? Les données suggèrent le scénario 2 sauf crise systémique majeure.

## §2 LES 2 SCÉNARIOS

### Scénario 1 : Le climat force la réforme

Coûts physiques (inondations, sécheresses, migrations) dépassent les profits → les élites intègrent le risque → taxation carbone/wealth → redistribution forcée. Précédent : WW2 a forcé le New Deal.

**Probabilité :** Faible à moyen terme. Les coûts sont externalisés sur le Sud global et les pauvres. Les riches s'adaptent (climatisation, assurances, déménagement).

### Scénario 2 : Le climat accélère la concentration

Les « stranded assets » (fossiles non exploitables) détruisent les petits porteurs → les riches rachètent à bas prix → concentration renforcée. Les terres habitables sont achetées par les ultra-riches → « climate gentrification ». L'adaptation est privatisée → ceux qui peuvent payer survivent.

**Probabilité :** Plus élevée. Le FDIC documente déjà les risques pour le système bancaire. Le WIL montre que le top 1% cause 41% des émissions — mais ce même top 1% sera le MIEUX protégé.

---

## §3 RÉPONSE À LA QUESTION FALSIFIANTE

**« Le climat pourrait-il casser le circuit ? »**

Oui, si une crise climatique suffisamment grave (perte de terres cultivables à grande échelle, migrations massives, effondrement de chaînes d'approvisionnement) force la main politique. Mais les données montrent que les coûts sont décalés : les pauvres subissent, les riches s'adaptent. Sans un événement qui frappe les riches DIRECTEMENT (perte de propriétés côtières de luxe, effondrement de marchés financiers), la réforme n'est pas forcée.
