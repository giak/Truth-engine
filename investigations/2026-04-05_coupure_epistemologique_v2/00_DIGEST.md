### 7 Angles Morts identifiés
1. **🔴 Polanyi** : Le "Paradoxe de l'Articulabilité" = paradoxe de Polanyi (1966). Omission de paternité.
2. **🔴 L'IA est tacite** : Les LLM fonctionnent par intuition statistique, pas par raisonnement explicite. Dichotomie fausse.
3. **🟡 Dimension corporelle** : L'intuition humaine est somatique. L'IA n'a pas de corps.
4. **🟡 Question de la confiance** : Deux intuitions tacites opposées → comment trancher ?
5. **🟡 Capture économique** : Les propriétaires de modèles capturent 99% de la valeur, pas les philosophes.
6. **🟡 Transmission** : Comment acquérir le "goût" ? Par immersion, pas par lecture.
7. **🔴 Contradiction performative** : L'article articule l'inarticulable. Il se détruit lui-même.

### Thèse locale
L'article v1 est brillant mais incomplet. 3 angles morts critiques nécessitent une réécriture profonde, pas des corrections cosmétiques.

### Connexions
- → Polanyi : fondement théorique manquant
- → Intuition LLM : dichotomie à reformuler
- → Corps : vraie différence humain/IA

### Contradictions internes
- L'audit lui-même est une tentative d'articuler l'inarticulable
- Les 7 angles morts sont eux-mêmes articulables → paradoxe récursif
