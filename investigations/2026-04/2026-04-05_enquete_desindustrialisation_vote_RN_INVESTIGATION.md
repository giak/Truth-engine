# ENQUÊTE — Désindustrialisation, vote RN et fracture politique française

**Date:** 2026-04-05
**Complexité:** COMPLEX
**Thème:** Vote RN, désindustrialisation, géographie électorale, populisme, déclassement

---

## §0 RÉSUMÉ EXÉCUTIF

Le lien entre désindustrialisation et vote RN est **documenté mais nuancé**. Anaïs Voy-Gillis (géographe, spécialiste industrie) établit un lien entre fermeture d'usines et vote nationaliste. L'IPP (Institut des politiques publiques, fév. 2026) publie une note sur les "effets locaux de la désindustrialisation". Alternatives Économiques (mars 2026) montre que **l'insatisfaction au travail est un facteur majeur du vote RN**. Trendeo note qu'il n'y a **pas de corrélation évidente** entre emploi industriel et vote RN — le lien est plus complexe que la simple présence d'usines. Le RN a publié son **propre rapport sur la désindustrialisation** (juillet 2025), instrumentalisant le sujet.

## §2 ANALYSE — Pourquoi la corrélation n'est pas simple

### 2.1 Le paradoxe Trendeo

Trendeo (cabinet qui suit les ouvertures/fermetures d'usines) note : **"Entre l'emploi industriel et le vote RN, pas de corrélation évidente."**

Pourquoi ? Parce que :
1. Le RN vote aussi dans des zones **non industrielles** (rurales, périurbaines)
2. Les zones industrielles restantes (Est, Alsace) votent **moins RN**
3. Le vote RN est **multifactoriel** : immigration, insécurité culturelle, déclin des services publics

### 2.2 La thèse Voy-Gillis

Anaïs Voy-Gillis (docteure en géographie, spécialiste industrie) défend une position **nuancée mais ferme** :
- La désindustrialisation crée un **terreau** (déclassement, perte de sens, isolement)
- Mais le vote RN n'est pas **mécanique** — il dépend d'autres facteurs
- "On interroge assez peu le rôle que peut avoir l'industrie dans notre société" (La French Fab, mars 2025)
- Elle publie "Pour une Révolution Industrielle" (lu par la Chaire Économie du Climat, mars 2025)

### 2.3 L'IPP — Les "laissés-pour-compte"

L'IPP (fév. 2026) publie la **Note n°123** : "Laissés-pour-compte : les effets locaux de la désindustrialisation en France."

Cette étude est **majeure** — elle mesure les effets LOCALISÉS de la désindustrialisation sur :
- Le tissu social
- Le vote
- Les services publics
- La démographie

## §3 CHAÎNES CAUSALES

### Chaîne 1 : Fermeture d'usine → Déclassement → Vote protestataire
Usine ferme → Chômage structurel → Perte de statut social → Sentiment d'abandon → Défiance envers élites → Vote RN (ou abstention)

### Chaîne 2 : Désindustrialisation → Désertification → Fracture territoriale
Usines ferment → Emploi disparaît → Services publics ferment (écoles, hôpitaux, gares) → Territoire se vide → Sentiment d'être "laissé-pour-compte" → Vote anti-système

### Chaîne 3 : Le RN capte le ressentiment industriel
Désindustrialisation réelle → RN publie un rapport → Instrumentalisation politique → "Seul le RN défend l'industrie" → Narratif Λ=9 → Vote RN

## §4 IMPACT VERDICT

**Qui gagne.** Le RN — la désindustrialisation est un argument électoral puissant. Les élus RN qui posent des questions au gouvernement sur les fermetures d'usines.

**Qui perd.** Les ouvriers licenciés — leur colère est captée politiquement sans que leur situation s'améliore. Les territoires industriels abandonnés.

**Qui meurt.** Le débat rationnel sur la politique industrielle — réduit à un enjeu électoral.

**Qui recule.** Le "bloc central" — qui regarde la désindustrialisation avec "bienveillance" (Le Monde) quand le RN en fait un sujet politique.

## §8 CONCLUSION PROVISOIRE

Le lien entre désindustrialisation et vote RN est **réel mais indirect**. La désindustrialisation crée un terreau de déclassement, d'abandon et de colère. Le RN (et d'autres populistes) **capture** cette colère avec des narratifs simplificateurs. Le tweet @LeBunkerBtc est une forme culturelle de cette même capture : il prend un ressentiment réel (le déclassement des diplômés) et le canalise vers un narratif faux mais émotionnellement satisfaisant ("le CAP gagne plus que le Master").

**La vraie question n'est pas "est-ce que la désindustrialisation fait voter RN" mais "qui capture le ressentiment créé par la désindustrialisation ?"**
