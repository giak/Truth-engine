# 🖼️ CLUSTER FRAMING - Narrative Control Patterns
*How the debate boundaries are set (5KB)*

## Parent: Λ (LAMBDA/FRAMING)
Activated when Λ score ≥5

## Related Concepts

### FALSE_DILEMMA
**Pattern**: Only two choices presented
**Detection**:
- "Either... or" framing
- Third option never mentioned
- Nuance eliminated
**Query boost**: "alternatives to", "third way", "other options"

### OVERTON_WINDOW
**Pattern**: Acceptable opinion boundaries
**Detection**:
- "Extreme" label for common sense
- "Moderate" label for actual extreme
- Discourse boundaries enforced
**Query boost**: "public opinion range", "acceptable discourse", "mainstream vs"

### ANCHORING_BIAS
**Pattern**: First number sets reference
**Detection**:
- Arbitrary starting point given
- All comparisons to that anchor
- Real baseline hidden
**Query boost**: "compared to what", "baseline year", "reference point"

### METAPHOR_CONTROL
**Pattern**: Metaphors shape thinking
**Detection**:
- War metaphors for non-war
- Market metaphors for non-market
- Natural metaphors for artificial
**Query boost**: "actually means", "literal definition", "not metaphor"

### QUESTION_FRAMING
**Pattern**: Questions determine answers
**Detection**:
- Leading questions dominant
- Assumptions built into query
- Real question never asked
**Query boost**: "underlying assumption", "real question", "presupposition"

### DEFINITION_GAMING
**Pattern**: Win by defining terms
**Detection**:
- Key terms redefined quietly
- Technical definition vs common
- Goalpost moving via definitions
**Query boost**: "definition of", "what counts as", "technically means"

### BOUNDARY_DRAWING
**Pattern**: In-group/out-group creation
**Detection**:
- "Us vs them" framing
- Arbitrary boundary lines
- Unity preventing divisions
**Query boost**: "who is included", "boundary between", "division created"

### EUPHEMISM_TREADMILL
**Pattern**: Language laundering reality
**Detection**:
- Harsh reality softened
- "Collateral damage" not "dead civilians"
- "Restructuring" not "mass layoffs"
**Query boost**: "really means", "plain language", "actual impact"

### NARRATIVE_MONOPOLY
**Pattern**: Single story dominates
**Detection**:
- One narrative everywhere
- Counter-narratives absent
- Story coordination visible
**Query boost**: "alternative narrative", "different perspective", "suppressed story"

### AGENDA_SETTING
**Pattern**: Control what's discussed
**Detection**:
- Some topics prominent
- Others mysteriously absent
- Timing of topic introduction
**Query boost**: "why now", "what's not discussed", "agenda behind"
