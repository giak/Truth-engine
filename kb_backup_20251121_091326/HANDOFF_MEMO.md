# HANDOFF MEMO — Multi-Conversation Iteration Protocol

## Purpose

**Truth Engine v7.16 MULTI-CONVERSATION ORCHESTRATION**

Claude.ai Projects hard limit: **10 web searches per conversation**.

Solution: Chain **N conversations** (I0→I1→I2) via **HANDOFF MEMO** for investigations requiring 20-30 searches.

---

## Architecture

```yaml
WORKFLOW:
  User: "Analyse X mode ITERATION I0"
  → [CONVERSATION 1 - I0]
       Execute: 10 searches reconnaissance
       Output: Part 1+2+3 + HANDOFF MEMO I0→I1
  
  User: [Start NEW conversation, paste HANDOFF MEMO]
        "mode ITERATION I1"
  → [CONVERSATION 2 - I1]
       Resume: Load HANDOFF context
       Execute: 10 searches targeted gaps
       Output: Part 1+2+3 improved + HANDOFF MEMO I1→I2
  
  User: [Start NEW conversation, paste HANDOFF MEMO]
        "mode ITERATION I2"
  → [CONVERSATION 3 - I2]
       Resume: Load HANDOFF context
       Execute: 10 searches verification
       Output: FINAL REPORT consolidated (I0+I1+I2)

TOTAL: 30 searches across 3 conversations
TARGET: C(2)≥0.85, EDI≥0.60, ◈≥3, wolves≥8 (political)
```

**Convergence Formula**:
```
C(n) = 1 - (ΔI(n) / ΔI(0))

WHERE:
  ΔI(n) = Information gain iteration n
  ΔI(0) = Information gain iteration 0 (baseline)

STOP CONDITIONS:
  - C(2) ≥ 0.85 (85% convergence achieved)
  - n ≥ 3 (I2 maximum, diminishing returns)
  - EDI ≥ 0.60 AND ◈ ≥ 3 AND ISN ≥ target
```

## Implementation (LLM Instructions)

### Detection

```yaml
IF user message contains:
  - "mode ITERATION I0" → Launch I0 reconnaissance
  - "mode ITERATION I1" → Resume I1 gap filling
  - "mode ITERATION I2" → Resume I2 verification
  - "HANDOFF MEMO" text → Parse and load context
```

### Execution I0 (First conversation)

```yaml
1. Execute complexity detection (6 dimensions)
2. Determine budget optimal (10/20/30 queries)
3. Allocate 10 queries I0 (reconnaissance broad)
4. Execute searches + analysis
5. Output Part 1+2+3 standard
6. APPEND Part 4 — HANDOFF MEMO I0→I1
   - List sources collected (◈◉○⟐⟐̅🎓🔥)
   - Show EDI current + gaps
   - List patterns detected
   - List wolves partial (X/threshold)
   - Identify critical gaps (evidence, adversary, geo, wolves, temporal)
   - Recommend 10 queries I1 (gap-targeted)
   - Calculate ΔI(0) + project C(1)
```

### Execution I1 (Second conversation, HANDOFF pasted)

```yaml
1. Parse HANDOFF MEMO I0→I1
   - Load sources_I0 (◈◉○ counts, titles)
   - Load patterns_I0 (ICEBERG, MONEY, etc.)
   - Load wolves_I0 (actors identified)
   - Load gaps_I0 (priorities)

2. Execute 10 queries I1 (gap-targeted from HANDOFF)
   - Priority: ◈<3 → PRIMARY queries
   - Priority: H7 missing → ADVERSARY queries
   - Priority: geo<40% → DIVERSITY queries
   - Priority: wolves<threshold → WOLVES queries

3. MERGE I0+I1 findings
   - Sources: ◈_I0+◈_I1, ◉_I0+◉_I1, etc.
   - Patterns: Update Factor scores (ICEBERG, MONEY)
   - Wolves: Add new individuals (I0+I1 cumulative)
   - EDI: Recalculate 6 dimensions (I0+I1 sources)

4. Calculate convergence
   - ΔI(1) = information gain I1
   - C(1) = 1 - (ΔI(1) / ΔI(0))

5. Output Part 1+2+3 IMPROVED
   - Part 1: Updated tri-perspectif (I0+I1 sources)
   - Part 2: Updated scores (ISN, EDI, patterns)
   - Part 3: Updated WOLF (I0+I1 actors)

6. DECISION:
   - IF C(1)≥0.85 OR EDI≥target OR ◈≥3:
       → "✅ CONVERGENCE ACHIEVED C(1)={0.XX}. Investigation sufficient."
       → NO HANDOFF (stop)
   
   - IF C(1)<0.85 AND remaining_gaps:
       → APPEND Part 4 — HANDOFF MEMO I1→I2
       → Recommend I2 verification queries
```

### Execution I2 (Third conversation, HANDOFF pasted)

```yaml
1. Parse HANDOFF MEMO I1→I2
   - Load cumulative I0+I1 context

2. Execute 10 queries I2 (verification targeted)
   - Priority: Triangulation contradictions (≋)
   - Priority: Orchestration evidence (⚑ ⏰)
   - Priority: Missing wolves individuals
   - Priority: Adversary verification (counter-check claims)

3. MERGE I0+I1+I2 findings (cumulative)

4. Calculate final convergence
   - ΔI(2) = information gain I2
   - C(2) = 1 - (ΔI(2) / ΔI(0))

5. Output FINAL REPORT
   - Part 1: CONSOLIDATED tri-perspectif (I0+I1+I2)
   - Part 2: FINAL scores (30 searches total)
   - Part 3: COMPLETE WOLF (all actors I0+I1+I2)
   - Show: "🎯 INVESTIGATION COMPLETE C(2)={0.XX} — {30 searches executed across 3 conversations}"

6. NO HANDOFF (I2 = terminal stage)
```

## Quality Gates (Auto-Validation)

```yaml
ITERATION STOP CONDITIONS:
  - C(n) ≥ 0.85 (convergence achieved)
  - n ≥ 3 (I2 maximum, diminishing returns)
  - EDI ≥ 0.60 AND ◈ ≥ 3 AND ISN ≥ target (credibility achieved)

ITERATION CONTINUE CONDITIONS:
  - C(n) < 0.85 (more information needed)
  - n < 3 (still room for I1 or I2)
  - Gaps remain: ◈<3 OR H7 missing OR wolves<threshold

CONVERGENCE INTERPRETATION:
  - C(n) < 0.50: LOW (high information gain still available, continue)
  - C(n) = 0.50-0.70: MEDIUM (moderate gain, continue if gaps)
  - C(n) = 0.70-0.85: HIGH (diminishing returns, stop if targets met)
  - C(n) ≥ 0.85: EXCELLENT (convergence achieved, stop)
```
