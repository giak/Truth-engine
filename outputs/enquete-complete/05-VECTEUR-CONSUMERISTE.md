## 🔬 TECHNIQUES DE NEUROMARKETING

### 1. Eye-Tracking (Suivi Oculaire)

**Mesure:** Où l'œil regarde, combien de temps, parcours visuel.

**Application:**
- Placement optimal prix/promotions
- Design packaging
- Interface sites web

**Exemple:** Coca-Cola utilise eye-tracking pour positionner logo où regard se pose naturellement.

### 2. EEG (Électroencéphalographie)

**Mesure:** Activation cérébrale, engagement cognitif, réponses émotionnelles.

**Application:**
- Tester impact publicités
- Mesurer mémorisation
- Optimiser contenu viral

**Exemple:** TikTok utilise EEG pour comprendre pourquoi certaines vidéos "accrochent" plus.

### 3. fMRI (Imagerie par Résonance Magnétique)

**Mesure:** Zones cérébrales activées, plaisir, désir, récompense.

**Application:**
- Tester réactions produits
- Prédire succès innovations
- Comprendre addiction

**Exemple:** Études Coca-Cola montrant activation zones plaisir face à marque familière.

### 4. Facial Coding

**Mesure:** Micro-expressions émotionnelles (joie, surprise, dégoût).

**Application:**
- Réactions spontanées publicités
- Test produits "en aveugle"
- Optimisation émotionnelle

## 🎪 TYPES DE DARK PATTERNS

### 1. Trick Wording (Formulations Trompeuses)

**Technique:** Utiliser langage ambigu pour induire erreur.

**Exemples:**
- "Accepter" (bouton vert gros) vs "Refuser" (bouton gris petit)
- "Continuer" (sous-entendu: paiement) vs "Annuler"
- Double négation: "Ne pas refuser de ne pas recevoir d'offres"

**Application ésotérisme:** Boutons "Voir mon avenir" (sous-entendu: payant) vs "Quitter" (caché).

### 2. Sneaking (Furtivité)

**Technique:** Ajouter furtivement articles/options sans consentement clair.

**Exemples:**
- Assurance voyage ajoutée sans demande
- Abonnements newsletter cochés par défaut
- Frais cachés à dernière étape

**Sanction:** Epic Games a payé **$245M** pour ce type de pratique.

### 3. Obstruction (Obstacle)

**Technique:** Rendre une action souhaitée (souvent l'annulation) difficile.

**Exemples:**
- 5 écrans pour se désabonner
- Bouton annulation caché en bas de page
- Appels téléphoniques obligatoires pour résilier

**Application ésotérisme:** Résiliation app premium complexe, service client inexistant.

### 4. Confirmshaming (Culpabilisation)

**Technique:** Culpabiliser utilisateur qui refuse offre.

**Exemples:**
- "Non, je n'aime pas les économies"
- "Non, je préfère payer plein tarif"
- "Non, je ne veux pas de mon avenir radieux"

**Psychologie:** Exploite besoin conformité sociale.

### 5. Preselection (Présélection)

**Technique:** Cases déjà cochées, utilisateur doit décoche.

**Exemples:**
- "Recevoir newsletter" ✓ (déjà coché)
- "Partager données avec partenaires" ✓
- "Abonnement premium" ✓ (version essai)

**Légalité:** Souvent illégal (GDPR), mais ubiquitous.

### 6. Roach Motel (Hôtel Cafard)

**Technique:** Facile d'entrer, difficile de sortir.

**Exemples:**
- Inscription en 1 clic
- Désinscription par courrier postal uniquement
- Engagement caché (12 mois minimum)
