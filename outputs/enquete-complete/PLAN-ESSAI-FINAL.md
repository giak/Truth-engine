## ACTE I : ORIGINE — Nous Sommes des Croyants Nés (~1 500 mots)

### I.1 La Vulnérabilité Biologique (600 mots)

**Concept :** Expliquer que la créduité n'est pas un bug, c'est une feature évolutive

**Plan détaillé :**

**A. Le HADD (250 mots)**
- ◈ Justin Barrett — Hyperactive Agency Detection Device
- Mécanisme évolutif : faux négatif = mort, faux positif = dépense énergétique
- Conséquence : nous voyons des agents intentionnels partout
- Exemple : grondement dans le noir = lion (même sans lion)

**B. L'enfant comme croyant naturel (250 mots)**  
- ◈ Bruce Hood — "SuperSense", "born believers"
- Les trois mécanismes :
  1. Essentialisme (essence invisible des objets)
  2. Causalité magique (actions symboliques influencent réalité)
  3. Dualisme corps/esprit (survie après mort)
- ◈ Subbotsky : les croyances sont "supprimées, pas éliminées" — réactivables sous stress

**C. Piaget et la pensée magique (100 mots)**
- Stade préopérationnel (2-7 ans) — pensée magique dominante
- Problème : beaucoup d'adultes ne dépassent jamais ce stade
- Mécanisme de régression : stress → retour stade préopérationnel

**D. Atran & Norenzayan : La Religion comme Spandrel (100 mots)**
- ◈ Atran & Norenzayan (2004) — "Religion's evolutionary landscape"
- Thèse : religion = "spandrel" (sous-produit de capacités cognitives)
- Théorie de l'esprit, causalité, détection d'agents → religion comme effet de bord
- **Implication :** on ne peut pas éliminer la croyance sans éliminer fonctions cognitives essentielles

**E. Ford : La Critique du HADD (100 mots)**
- ◈ Kat Ford (2024) — "HADD its day"
- Critique : aucune preuve d'un "dispositif hérité" spécifique
- Mais : propension cross-culturelle à la créduité reste documentée
- Conclusion : mécanisme exact contesté, vulnérabilité réelle

**F. Lacan : Fondateurs vs Adeptes (100 mots)**
- ◈ Lacan — distinction psychotique vs névrotique
- **Fondateurs** de religions/croyances → Psychotiques (créateurs systèmes délirants)
- **Adeptes** → Névrosés (besoin structure symbolique)
- Application : compréhension des gourous (fondateurs) vs victimes (adeptes)

**Citation d'accroche :**
> "Nous sommes des croyants nés. La pensée magique est câblée dans nos cerveaux." — Bruce Hood

**Concept :** Le modèle ternaire qui ne changera jamais

**A. Babylone (3000 BP) — L'astrologie savante (200 mots)**

- Pouvoir de prédiction = pouvoir politique

**B. L'oracle de Delphes — Le modèle parfait (200 mots)**
- Structure :
  1. Données (astres/entrailles)
  2. Traitement opacifié (calculs/interprétation)  
  3. Prédiction ambiguë (toujours raison rétrospectivement)
- Cette structure ne changera jamais — seul le support change

**C. Le schéma structurel (150 mots)**
```
DONNÉES → TRAITEMENT OPACIFIÉ → PRÉDICTION AMBIGUË
```
- **Analogique (antique)** : Astres → Prêtres → Prophéties
- **Algorithmique (2025)** : Données utilisateur → Algorithmes ML → "Insights" personnalisés
- Même mécanisme psychologique : besoin de contrôle face à l'incertitude
- **Ce schéma ternaire sera le fil conducteur de tout l'essai**
