# PROMPT — ILLUSTRATION ESSAI "L'EMPIRE DU MENSONGE"

## Contexte de l'Essai

**Titre** : L'EMPIRE DU MENSONGE : Enquête sur la Guerre Souterraine de l'Évolution

**Thème central** : Le débat évolution/création n'est pas un débat scientifique — c'est une guerre culturelle politico-économique où la vérité est la première victime. Au-delà de cette guerre, il existe une science réelle de l'évolution.

**Éléments clés à illustrer** :
- La guerre souterraine (guerre culturelle, économique, politique)
- Les fraudes scientifiques persistantes (Haeckel, Peppered Moth, Miller-Urey)
- L'iceberg (ICEBERG MAX analysis — 7 dimensions)
- Deux camps qui s'affrontent dans l'ombre
- La vérité enfouie, les manipulations en surface
- Dollars, manuels scolaires, embryons, papillons

**Ambiance** : Sombre, conspiration, guerre froide, atlas mythological
