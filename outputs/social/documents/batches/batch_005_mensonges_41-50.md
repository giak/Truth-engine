## Mensonge #41

### Description
**Déclaration** : "Nous n'avons pas menti sur les masques" (Olivier Véran)  
**Date** : 2020  
**Réalité** : Stock stratégique de masques non renouvelé depuis 2013, pénurie avérée

### Type de mensonge
**MENSONGE d'État** - Déni face aux preuves

### Contexte
Olivier Véran, ministre de la Santé, nie tout mensonge sur les masques. Le Rapport du Sénat sur la gestion Covid révèle que le stock stratégique (1 milliard de masques en 2010) n'a jamais été renouvelé depuis 2013. En mars 2020, la France n'avait que quelques millions de masques pour 67 millions d'habitants.

### Sources
- **Type** : ◈ Rapport Sénat (enquête Covid)
- **Lien** : Commission d'enquête Sénat 2020
- **Stocks** : Données ministère Santé

### Informations complémentaires
- Stock 2010 : 1 milliard masques (post-grippe H1N1)
- Stock 2020 : quelques millions (périmés)
- Non-renouvellement : 2013-2020
- Pénurie : personnels soignants sans protection
- Commandes d'urgence : retards, prix multipliés par 10
- Rapport Sénat : responsabilités établies

## Mensonge #42

### Description
**Déclaration** : "Un plan massif d'investissement pour l'hôpital"  
**Date** : 2020 (Ségur de la Santé)  
**Réalité** : Le Ségur = rattrapage de la dette hospitalière, pas d'investissement net

### Type de mensonge
**Λ (FRAMING)** - Présenter un rattrapage de dette comme un investissement

### Contexte
Le "Ségur de la Santé" (2020) annonce 19 milliards d'euros pour l'hôpital. Les analyses budgétaires montrent que l'essentiel sert à éponger la dette accumulée des hôpitaux (déficits structurels), pas à créer de nouveaux moyens (lits, personnel, équipements). Un tour de passe-passe comptable.

### Sources
- **Type** : ◉ Analyses budgétaires (syndicats, économistes)
- **Ségur** : ◈ Annonces officielles vs réalité terrain
- **Dette hôpitaux** : ◈ Données FHF (Fédération Hospitalière)

### Informations complémentaires
- Annonce : 19 Md€ Ségur
- Réalité : majorité pour dette/arriérés
- Investissement net : marginal
- Lits créés : quasi nuls
- Personnel : primes mais pas recrutements massifs
- Soignants : sentiment de duperie

## Mensonge #43

### Description
**Déclaration** : [Pas de déclaration]  
**Date** : 2017-2019  
**Réalité** : 5 700 lits d'hôpital supprimés

### Type de mensonge
**SUPPRESSION** - Action inverse des promesses

### Contexte
Malgré les promesses de renforcer l'hôpital public, 5 700 lits ont été supprimés entre 2017 et 2019, avant même le Covid. Cette politique de réduction continue aggrave l'engorgement des urgences et les délais de prise en charge. Les soignants dénoncent une gestion comptable au détriment des patients.

### Sources
- **Type** : ◈ SAE (Statistique Annuelle Établissements)
- **Données** : DREES, ministère Santé
- **Suivi** : ◉ Syndicats soignants

### Informations complémentaires
- 2017-2019 : -5 700 lits
- Logique : T2A (tarification à l'activité) pousse à fermer lits "non rentables"
- Conséquences : délais attente, déprogrammations
- Covid : pénurie lits réa aggravée par ces suppressions
- 2022-2024 : encore -10 000 lits

## Mensonge #44

### Description
**Déclaration** : "Les soignants sont les héros de la nation"  
**Date** : 2020 (pendant Covid)  
**Réalité** : Prime ponctuelle, puis rien de structurel (salaires, conditions, recrutements)

### Type de mensonge
**ABANDON** - Hommages sans actes

### Contexte
Durant le Covid, Macron qualifie les soignants de "héros". Applaudissements aux fenêtres à 20h, discours larmoyants. Une fois la crise passée : prime de 183€ à 1500€ selon catégories, puis plus rien. Pas de revalorisation salariale pérenne, pas de recrutements massifs, conditions de travail toujours dégradées.

### Sources
- **Type** : ◈ Suivi mesures post-Covid
- **Primes** : Montants et conditions
- **Salaires** : ◉ Comparaisons internationales (France sous moyenne OCDE)

### Informations complémentaires
- "Héros" pendant crise
- Prime "Ségur" : 183€ à 1500€ (selon statuts)
- Salaires infirmiers France : -7% moyenne OCDE
- Recrutements : insuffisants (déserts médicaux)
- Démissions : record en 2022-2024
- Sentiment : instrumentalisation puis abandon

## Mensonge #45

### Description
**Déclaration** : Proposition de "médaille de l'engagement" pour les soignants  
**Date** : 2020  
**Réalité** : Médaille reçue comme une insulte par les soignants

### Type de mensonge
**MÉPRIS** - Symbole à la place de mesures concrètes

### Contexte
Plutôt que des revalorisations salariales ou des recrutements, le gouvernement propose une "médaille de l'engagement" aux soignants ayant combattu le Covid. Cette annonce provoque un tollé : les soignants demandent des moyens, pas des médailles. Symbole du décalage gouvernement/terrain.

### Sources
- **Type** : ◉ Réactions soignants (réseaux sociaux, syndicats)
- **Annonce** : Médaille de l'engagement (2020)
- **Analyses** : ◉ Médias, syndicats

### Informations complémentaires
- Annonce : création médaille
- Réactions : #pasdesapplaudissementsdesmoyens
- Soignants : colère, sentiment mépris
- Comparaison : autres pays revalorisent salaires
- France : médailles mais pas euros
- Symbole : communication vs action

- Réalité fin 2024 : toujours saturées
- Attentes : 8-12h moyennes zones tendues
- Personnels : burn-out, démissions
- Plans annoncés : sans effet
- Problème structurel non résolu

## Mensonge #47

### Description
**Déclaration** : "600 000 patients en ALD auront un médecin traitant d'ici fin 2022"  
**Date** : 2022  
**Réalité** : Promesse non tenue, déserts médicaux aggravés

### Type de mensonge
**FAUX** - Objectif chiffré non atteint

### Contexte
Les patients en ALD (Affection Longue Durée - maladies chroniques graves) sans médecin traitant devaient tous en avoir un fin 2022. L'objectif n'a pas été atteint. Les déserts médicaux se sont même aggravés : 6 millions de Français sans médecin traitant en 2024.

### Sources
- **Type** : ◉ Bilan objectifs santé
- **Données** : Ordre des médecins, ARS
- **Déserts médicaux** : Cartographies officielles

### Informations complémentaires
- Objectif : 600K patients ALD avec médecin fin 2022
- Réalité : objectif non atteint
- 2024 : 6 millions Français sans médecin traitant
- Déserts médicaux : 1/3 du territoire
- Démographie médicale : départs > arrivées
- Solutions : insuffisantes (maisons santé, téléconsultation limitée)

## Mensonge #48

### Description
**Déclaration** : "Pas de fermeture de maternités"  
**Date** : 2017  
**Réalité** : Dizaines de maternités fermées sur le quinquennat

### Type de mensonge
**MENSONGE factuel** - Fermetures effectives malgré promesse

### Contexte
Le gouvernement s'était engagé à ne pas fermer de maternités. En réalité, des dizaines de maternités ont fermé pour "raisons de sécurité" (seuils de naissances non atteints). Les femmes enceintes doivent parcourir des dizaines de kilomètres, avec risques d'accouchements en urgence.

### Sources
- **Type** : ◈ Carte des maternités (ministère Santé)
- **Fermetures** : Listes établissements
- **Témoignages** : ◉ Collectifs citoyens, élus locaux

### Informations complémentaires
- Promesse : maintien maternités
- Réalité : dizaines de fermetures
- Critère : seuil 300 naissances/an
- Conséquences : trajets allongés (danger)
- Accouchements urgence : en augmentation
- Territoires ruraux : désertification médicale

## Mensonge #49

### Description
**Déclaration** : "On recrute massivement des soignants"  
**Date** : 2023  
**Réalité** : Hémorragie de personnel, démissions record

### Type de mensonge
**Ω (INVERSION)** - Annonce recrutements, réalité : départs massifs

### Contexte
Face à la crise hospitalière, le gouvernement annonce des recrutements. Dans les faits, les hôpitaux subissent une hémorragie : démissions record d'infirmiers, médecins hospitaliers partant vers le libéral, burn-out généralisé. Plus de départs que d'arrivées.

### Sources
- **Type** : ◈ Statistiques hospitalières (DREES)
- **Démissions** : Données RH hôpitaux
- **Analyses** : ◉ Syndicats (CGT Santé, SUD)

### Informations complémentaires
- Annonces : recrutements massifs
- Réalité : solde négatif (départs > arrivées)
- Démissions infirmiers : +40% 2020-2024
- Médecins hospitaliers : fuite vers libéral
- Burn-out : 60% soignants concernés
- Attractivité : salaires faibles, conditions dégradées

## Mensonge #50

### Description
**Déclaration** : "On investit dans la santé mentale"  
**Date** : 2022  
**Réalité** : Moyens insuffisants, CMP (Centres Médico-Psychologiques) saturés

### Type de mensonge
**FAUX** - Annonces sans moyens réels

### Contexte
Des "Assises de la santé mentale" sont organisées, des plans annoncés. Sur le terrain : CMP saturés (délais de 6 mois à 1 an), pédopsychiatrie en crise, lits psy fermés. Les moyens annoncés ne se matérialisent pas ou sont largement insuffisants face aux besoins.

### Sources
- **Type** : ◉ Rapports terrain (associations, syndicats psy)
- **Assises** : 2021-2022
- **Saturation** : Témoignages patients, professionnels

### Informations complémentaires
- Assises santé mentale : 2021-2022
- Budget annoncé : 1,28 Md€ sur 5 ans
- Réalité terrain : CMP saturés, délais 6-12 mois
- Pédopsy : crise majeure (suicides ados)
- Lits psychiatrie : fermetures continues
- Besoin vs moyens : décalage abyssal
