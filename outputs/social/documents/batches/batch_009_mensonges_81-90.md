## Mensonge #81

### Description
**Déclaration** : [Pas de mention dans communication]  
**Date** : 2018  
**Réalité** : 45% des smicards exclus de la prime d'activité (conditions revenus du foyer)

### Type de mensonge
**OMISSION** - Ne pas préciser que près de la moitié ne touchera rien

### Contexte
Quand Macron annonce "100€ de plus pour les smicards", il omet de préciser que la prime d'activité est soumise à conditions de ressources du FOYER (pas seulement individu). Résultat : 45% des smicards n'y ont pas droit car leur conjoint(e) travaille ou revenus foyer trop élevés. Près de la moitié des concernés exclus!

### Sources 
- **Type** : ◈ Conditions décret application
- **Exclus** : ~45% smicards (calcul CAF)
- **Raison** : Revenus foyer, pas individu

### Informations complémentaires
- Prime activité : conditions ressources FOYER
- Smic individuel : pas suffisant, compte conjoint
- 45% smicards exclus
- Annonce "tous smicards" : fausse
- Complexité administrative : beaucoup ne demandent pas
- Taux non-recours : ~30% ayants-droit

## Mensonge #82

### Description
**Déclaration** : "Mesure sans coût pour l'employeur"  
**Date** : 2018  
**Réalité** : Donc pas une augmentation de salaire, juste aide sociale

### Type de mensonge
**Λ (FRAMING)** - Présenter aide sociale comme hausse salaire

### Contexte
Macron met en avant que la mesure est "sans coût pour l'employeur", comme un avantage. C'est surtout la preuve que ce N'EST PAS une hausse de salaire. Le SMIC horaire reste identique, l'employeur ne paie pas plus. C'est l'État (= contribuables) qui compense via aide sociale. Pas de revalorisation du travail.

### Sources
- **Type** : ◈ Mécanisme prime activité
- **Coût employeur** : 0€
- **Finance** : État (contribuables)

### Informations complémentaires
- SMIC horaire : inchangé
- Coût salarial employeur : inchangé
- Financement : budget État (CAF)
- Nature : aide sociale, pas salaire
- Pas de revalorisation travail
- Charge : collectivité, pas entreprises

## Mensonge #84

### Description
**Déclaration** : "Défiscalisation heures supplémentaires = mesure nouvelle"  
**Date** : 2018  
**Réalité** : Existait sous Sarkozy (2007-2012), supprimée par Hollande, rétablie

### Type de mensonge
**FAUX** - Recycler mesure passée en la présentant comme nouvelle

### Contexte
La défiscalisation des heures supplémentaires avait été instaurée par Sarkozy (2007), supprimée par Hollande (2012), et rétablie par Macron (2018). Ce n'est pas une "mesure nouvelle", c'est une résurrection d'une mesure de 11 ans. L'historique est omis pour donner impression d'innovation.

### Sources
- **Type** : ◈ Historique lois fiscales
- **Sarkozy** : 2007-2012 (défiscalisation HS)
- **Hollande** : 2012 (suppression)
- **Macron** : 2018 (rétablissement)

### Informations complémentaires
- 2007 : Sarkozy instaure défiscalisation HS
- 2012 : Hollande supprime (coût budgétaire)
- 2018 : Macron rétablit
- "Nouvelle" : faux, recyclage mesure Sarkozy
- Coût : 2-3 Md€/an (budget État)
- Efficacité : contestée (heures déjà faites déclarées)

## Mensonge #85

### Description
**Déclaration** : "Le pouvoir d'achat est protégé"  
**Date** : 2020  
**Réalité** : Inflation non compensée, perte pouvoir d'achat 2021-2024

### Type de mensonge
**FAUX** - Inflation explose, salaires stagnent

### Contexte
Macron affirme protéger le pouvoir d'achat. L'inflation 2021-2024 (énergie, alimentaire) explose : +15% cumulé. Les salaires ne suivent pas (+5-7% en moyenne). Résultat : perte de pouvoir d'achat de 5 à 8% selon catégories. L'INSEE confirme l'érosion généralisée malgré "boucliers" (prix limités).

### Sources
- **Type** : ◈ Données INSEE (inflation vs salaires)
- **Inflation** : +15% cumulé 2021-2024
- **Salaires** : +5-7% moyenne
- **Écart** : -5 à -8% pouvoir achat

### Informations complémentaires
- Inflation 2021-2024 : +15% (alimentaire +20%)
- Salaires réels : +5-7%
- Perte nette : -5 à -8%
- Bouclier tarifaire : insuffisant
- Essence, gaz, courses : explosion prix
- "Protégé" : faux, érosion massive

## Mensonge #86

### Description
**Déclaration** : "Les chèques énergie sont suffisants"  
**Date** : 2022  
**Réalité** : Explosion des factures d'énergie malgré les aides, précarité énergétique

### Type de mensonge
**FAUX** - Aides insuffisantes face à explosion prix

### Contexte
Le gouvernement distribue des chèques énergie (100-200€) présentés comme solution à l'explosion des prix gaz/électricité. Les factures annuelles augmentent de 500 à 1000€. Les chèques compensent 10-20% de la hausse. Précarité énergétique explose : 12 millions de Français en difficulté paiement énergie.

### Sources
- **Type** : ◉ Témoignages, associations (Fondation Abbé Pierre)
- **Aides** : 100-200€
- **Hausses** : 500-1000€/an
- **Gap** : Aides couvrent 10-20% hausse

### Informations complémentaires
- Chèques énergie : 100-200€
- Hausse factures : 500-1000€/an
- Compensation : 10-20% seulement
- 12 millions Français précarité énergétique
- Impayés : explosent 2022-2024
- Coupures : en augmentation
