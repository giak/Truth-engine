## Mensonge #91

### Description
**Déclaration** : "Fin du cumul des mandats"  
**Date** : 2017  
**Réalité** : Nombreux contournements de la loi via fonctions de "délégués", présidences d'intercommunalités

### Type de mensonge
**FAUX** - Loi contournée systématiquement

### Contexte
La loi anti-cumul de 2014 (maintenue par Macron) interdit d'être député/sénateur et maire/président département simultanément. Mais les contournements sont légion : maires deviennent "maires délégués", députés président d'intercommunalités (non comptabilisé comme mandat), etc. La loi est respectée formellement mais vidée de sa substance.

### Sources
- **Type** : ◉ Observations terrain (médias locaux)
- **Exemples** : Multiples cas documentés
- **Loi** : ◈ Loi anti-cumul 2014

### Informations complémentaires
- Loi 2014 : interdiction cumul député/maire
- Contournements : "maire délégué", président interco
- Interco : non comptée comme mandat (vide juridique)
- Résultat : esprit loi non respecté
- Cumul déguisé : fréquent
- Renouvellement politique : limité

## Mensonge #92

### Description
**Déclaration** : "Le Grand Débat = vraie écoute des Français"  
**Date** : 2019 (post-Gilets Jaunes)  
**Réalité** : Mesures adoptées sont souvent contraires aux cahiers de doléances citoyens

### Type de mensonge
**Ω (INVERSION)** - Faire l'inverse des demandes exprimées

### Contexte
Le Grand Débat national (2019) récolte 1,5 million de contributions citoyennes via cahiers de doléances, réunions, plateforme en ligne. Les demandes principales : plus de justice fiscale (taxes sur les riches), services publics renforcés, référendums. Les mesures prises vont souvent à l'inverse : taxe carbone retirée mais pas ISF, services publics maintenus a minima.

### Sources
- **Type** : ◉ Bilan Grand Débat (synthèse indépendante)
- **Contributions** : 1,5 M participations
- **Analyses** : ◉ ONG, syndicats

### Informations complémentaires
- 1,5 million contributions
- Demandes : ISF, référendums, services publics
- Réponses : ISF non rétabli, 0 référendum, services stagnent
- Taxe carbone : retirée (bien)
- Mais : demandes structurelles ignorées
- "Écoute" : sélective, oriente vers réponses voulues

## Mensonge #95

### Description
**Déclaration** : [Promesse implicite usage modéré 49.3]  
**Date** : 2017-2024  
**Réalité** : 23 recours à l'article 49.3 - record de la Ve République

### Type de mensonge
**RECORD / ABUS** - Usage constitutionnel mais abusif d'une procédure d'exception

### Contexte
L'article 49.3 permet de faire passer une loi sans vote, sauf motion de censure. C'était une procédure d'exception. Macron l'utilise 23 fois en 7 ans, dont 11 fois pour la seule réforme des retraites. Record absolu de la Ve République. Dérive vers un usage systématique pour contourner le Parlement.

### Sources
- **Type** : ◈ Comptage Assemblée nationale
- **Total** : 23× (2017-2024)
- **Retraites** : 11× à elle seule

### Informations complémentaires
- 23 recours au 49.3 (Macron)
- Dont 11× réforme retraites
- Record Ve République pour 1 président
- Comparaison : Valls 6× (2014-2016)
- Détournement : procédure exception devenue routine
- Opposition : impuissante

## Mensonge #98

### Description
**Déclaration** : "Le terme 'violences policières' n'existe pas juridiquement"  
**Date** : 2020  
**Réalité** : La qualification est reconnue par la jurisprudence, les tribunaux l'utilisent

### Type de mensonge
**FAUX juridique** - Mentir sur l'existence juridique d'un terme utilisé par les juges

### Contexte
Castaner puis Darmanin affirment que le terme "violences policières" n'existe pas dans le droit. C'est faux : les tribunaux utilisent cette qualification dans leurs jugements, condamnent des policiers pour "violences", et la CEDH emploie le terme. Jouer sur les mots pour nier une réalité juridiquement établie.

### Sources
- **Type** : ⚖️ Jurisprudence (jugements)
- **Tribunaux** : Utilisent qualification "violences policières"
- **CEDH** : Emploie le terme

### Informations complémentaires
- Tribunaux français : condamnent policiers pour violences
- Termes jugements : "violences illégitimes", "violences policières"
- CEDH : terme utilisé
- Jurisprudence : établie
- "N'existe pas juridiquement" : faux
- Sémantique pour nier réalité
