## Mensonge #101

### Description
**Déclaration** : Les Gilets Jaunes sont des "casseurs"  
**Date** : 2018-2019  
**Réalité** : Des milliers de manifestants pacifiques blessés, dont de nombreux non-casseurs

### Type de mensonge
**Λ (FRAMING)** - Qualifier tout un mouvement par ses extrémités

### Contexte
Le gouvernement qualifie régul

ièrement les Gilets Jaunes de "casseurs", amalgamant l'ensemble du mouvement avec les violences marginales. Pourtant, les chiffres montrent des milliers de blessés parmi les manifestants pacifiques : 30+ éborgnés par LBD, 5 mains arrachées par grenades, centaines de blessés graves. La violence policière a touché majoritairement des manifestants non-violents.

### Sources
- **Type** : ◈ Décompte blessés (médias indépendants, Désarmons-les)
- **Chiffres** : 30+ éborgnés, 5 mains, 1000+ blessés
- **Nature** : Médicalement constatés

### Informations complémentaires
- 30+ éborgnés par LBD
- 5 mains arrachées grenades GLI-F4
- Plus de 1000 blessés graves
- Majorité : manifestants pacifiques
- "Casseurs" : minorité utilisée pour discréditer ensemble
- Répression massive mouvement social

## Mensonge #102

### Description
**Déclaration** : Le maintien de l'ordre est "proportionné"  
**Date** : 2019  
**Réalité** : Techniques de nasse jugées illégales par le Conseil d'État

### Type de mensonge
**⚖️ CONTREDIT PAR JUSTICE** - Pratiques déclarées illégales

### Contexte
Le gouvernement défend le maintien de l'ordre comme "proportionné et adapté". Le Conseil d'État juge certaines techniques illégales : nasses (encerclement manifestants sans sortie pendant des heures), usage LBD dans certaines conditions, grenades de désencerclement. Les juges reconnaissent des atteintes disproportionnées aux libertés.

### Sources
- **Type** : ⚖️ Décisions Conseil d'État
- **Lien** : Ordonnances 2019-2020
- **Techniques** : Nasses, LBD, grenades

### Informations complémentaires
- Conseil d'État : techniques illégales identifiées
- Nasses : encerclement prolongé sans issue
- LBD : usage restreint (mais continue)
- Grenades désencerclement : danger reconnu
- Maintien ordre : qualifié disproportionné
- France condamnée internationalement

## Mensonge #103

**Déclaration** : Les émeutes 2023 sont la "faute des réseaux sociaux"  
**Date** : Juin-juillet 2023  
**Réalité** : Causes sociales (discrimination, chômage jeunes, quartiers) largement ignorées

**OMISSION** - Pointer bouc émissaire technologique, ignorer racines sociales

Après l'assassinat de Nahel par un policier (juin 2023), des émeutes éclatent. Le gouvernement pointe les "réseaux sociaux" comme responsables. Blâmer TikTok permet d'éviter débat de fond.

### Sources
- **Type** : ◉ Analyses sociologues (Mouhanna, Mucchielli)
- **Causes** : Chômage, discrimination, relations police
- **Données** : ◈ INSEE (chômage jeunes quartiers)

### Informations complémentaires
- Nahel : tué par policier 27/06/2023
- Émeutes : juin-juillet 2023
- Gouvernement : "faute réseaux sociaux"
- Sociologues : causes structurelles ignorées
- Chômage jeunes quartiers : 20-30%
- Discriminations : rapport Défenseur droits

## Mensonge #104

### Description
**Déclaration** : "Rétablissement de la police de proximité"  
**Date** : 2017  
**Réalité** : Jamais véritablement rétablie

### Type de mensonge
**FAUX / ABANDON** - Promesse non tenue

### Contexte
Macron promet le retour de la "police de proximité" (îlotage, contact population) supprimée sous Sarkozy. Des expérimentations locales sont menées, mais aucun déploiement national. Les effectifs manquent, les moyens aussi. La police reste organisée en intervention/patrouille motorisée, pas en proximité pérenne avec les habitants.

### Sources
- **Type** : ◉ Bilan terrain (syndicats, élus locaux)
- **Promesse** : Programme 2017
- **Réalité** : Expérimentations limitées

### Informations complémentaires
- Police proximité : supprimée Sarkozy (2003)
- Macron : promet retour (2017)
- Réalité : expérimentations locales sporadiques
- Pas de déploiement national
- Effectifs : insuffisants
- Organisation intacte : intervention motorisée

- **Justice** : ⚖️ Conseil d'État (restrictions)
- **ONU** : Rapport critique

### Informations complémentaires
- 30+ éborgnés confirmés
- Fractures crânes, traumas faciaux
- Défenseur droits : demande interdiction
- Conseil d'État : restrictions usage
- ONU : dénonce
- Gouvernement : maintient autorisation

## Mensonge #108

### Description
**Déclaration** : "Suppression ISF pour créer richesse et emplois" (ruissellement)  
**Date** : 2017  
**Réalité** : 5 milliards aux plus riches, aucune preuve de ruissellement

### Type de mensonge
**Ω (INVERSION)** - Promesse emplois, résultat : cadeau fiscal sans retour

### Contexte
L'ISF (Impôt Solidarité Fortune) rapportait 5 Md€/an. Macron le remplace par IFI (Immobilier uniquement), perdre 3,5 Md€/an. Argument : "ruissellement", les riches investiront, créeront emplois. France Stratégie et IPP (Institut Politiques Publiques) : aucune preuve de ruissellement. L'argent part en placements financiers, pas en économie réelle.

### Sources
- **Type** : ◉ Rapports IPP, France Stratégie
- **Perte** : 3,5 Md€/an
- **Ruissellement** : Non prouvé

### Informations complémentaires
- ISF : 5 Md€/an
- Remplacement : IFI (immobilier seulement)
- Perte : 3,5 Md€/an
- Bénéficiaires : 0,1% les plus riches
- Ruissellement : jamais prouvé économiquement
- IPP : argent va en placements, pas emplois

## Mensonge #109

### Description
**Déclaration** : "Je ne suis ni de droite ni de gauche"  
**Date** : 2017  
**Réalité** : Flat tax, ISF, loi travail = mesures droite/ultra-libérales

### Type de mensonge
**Λ (FRAMING)** - Se positionner "ni-ni" tout en menant politique de droite

### Contexte
Macron se présente en "dépassement clivages droite-gauche". Dans les faits : suppression ISF, flat tax 30% (cadeaux riches), ordonnances travail (faciliter licenciements), baisse impôt sociétés. Toutes mesures de droite économique classique. Le "ni-ni" est un marketing politique contredit par les actes.

### Sources
- **Type** : ◈ Bilan législatif
- **Mesures** : ISF, flat tax, loi travail, IS
- **Analyse** : ◉ Économistes (Piketty, etc.)

### Informations complémentaires
- ISF supprimé : droite (réclame depuis 30 ans)
- Flat tax 30% : droite (taxation capital = travail)
- Loi travail : droite (facilite licenciements)
- Impôt sociétés : baissé
- "Ni droite ni gauche" : slogan marketing
- Politique réelle : droite économique

## Mensonge #110

### Description
**Déclaration** : "Pas de nouveaux impôts"  
**Date** : 2019  
**Réalité** : Hausse CSG retraités (+1,7 point), taxes multiples

### Type de mensonge
**FAUX** - Hausse fiscale massive niée

### Contexte
Macron nie augmenter les impôts. Dès 2018, la CSG des retraités passe de 6,6% à 8,3% (+1,7 point), touchant 60% des retraités. Taxes carbone, tabac, essence, sont augmentées. La charge fiscale globale reste stable voire augmente, malgré baisses pour les riches (ISF). Les classes moyennes/modestes paient plus.

### Sources
- **Type** : ◈ Lois finances successives
- **CSG** : +1,7 point (2018)
- **Autres taxes** : Carbone, tabac

### Informations complémentaires
- CSG retraités : +1,7 point (2018)
- 60% retraités touchés
- Taxe carbone : hausses (avant retrait)
- Tabac, essence : augmentations
- Charge fiscale : stable/hausse pour classes moyennes
- "Pas nouveaux impôts" : faux factuel
