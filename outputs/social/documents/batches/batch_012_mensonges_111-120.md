## Mensonge #111

### Description
**Déclaration** : "Baisse dépenses publiques"  
**Date** : 2017-2024  
**Réalité** : Dette record +1 198 milliards d'euros depuis 2017

### Type de mensonge
**Ω (INVERSION)** - Promettre baisse, obtenir explosion

### Contexte
Macron promet de "maîtriser les dépenses publiques" et "réduire la dette". Bilan 2017-2025 : dette passe de 2 218 Md€ (96,8% PIB) à 3 416 Md€ (115,6% PIB). Augmentation de 1 198 milliards en 7 ans (+54%). Record historique absolu. L'inverse total de la promesse.

### Sources
- **Type** : ◈ Données INSEE
- **2017** : 2 218 Md€ (96,8% PIB)
- **T2 2025** : 3 416 Md€ (115,6% PIB)
- **Lien** : https://www.insee.fr

### Informations complémentaires
- Dette 2017 : 2 218 Md€
- Dette T2 2025 : 3 416 Md€
- Augmentation : +1 198 Md€ (+54%)
- PIB : 96,8% → 115,6%
- Record historique France
- "Baisse dépenses" : totalement faux

### Informations complémentaires

- Dette avant Covid : 98% PIB
- Dette 2025 : 115,6% PIB
- Différence : +17,6 points PIB permanents
- "Temporaire" : faux, structurel

**Déclaration** : "Le plein emploi est pour bientôt"  
**Date** : 2022  
**Réalité** : Chômage réel sous-estimé (halo chômage, découragés non comptés)

**Λ (FRAMING)** - Manipuler statistiques pour embellir

Macron annonce "plein emploi bientôt" avec chiffres officiels (~7% chômage). Chômage réel estimé 10-12% si on inclut toutes personnes cherchant emploi. Les statistiques sont trompeuses.

### Sources
- **Type** : ◉ Analyses alternatives (DARES, études)
- **Chômage officiel** : ~7%
- **Chômage réel** : 10-12% (avec halo)

### Informations complémentaires
- Chômage BIT : ~7% (critères restrictifs)
- Halo chômage : 1,7 M personnes
- Découragés : non comptés
- Chômage partiel : exclus stats
- Chômage réel : 10-12%
- "Plein emploi" : manipulation statistique

## Mensonge #116

### Description
**Déclaration** : "Le CICE a créé des emplois"  
**Date** : 2017  
**Réalité** : Effet emploi du CICE contesté par économistes

### Type de mensonge
**FAUX** - Efficacité non prouvée

### Contexte
Le CICE (Crédit Impôt Compétitivité Emploi) coûte 20 Md€/an à l'État. Macron en hérite (Hollande) et le transforme en baisses charges. Il affirme que cela crée emplois. Les économistes (France Stratégie, CAE) : effet très faible voire nul. L'argent part en marges des entreprises, pas en embauches.

### Sources
- **Type** : ◉ Rapports France Stratégie, CAE
- **Coût** : 20 Md€/an
- **Emplois créés** : 100-200K (optimiste) vs 20 Md€

### Informations complémentaires
- CICE : 20 Md€/an crédit impôt
- Objectif : emplois + compétitivité
- Résultat : 100-200K emplois (France Stratégie, optimiste)
- ROI : 100-200K€ par emploi créé (très cher)
- Argent : majorité en marges entreprises
- Efficacité : très contestée

## Mensonge #117

### Description
**Déclaration** : "La flat tax encourage l'investissement productif"  
**Date** : 2017  
**Réalité** : Flat tax 30% facilite surtout l'optimisation fiscale, pas investissement réel

### Type de mensonge
**FAUX** - Objectif affiché vs usage réel

### Contexte
La "flat tax" (PFU - Prélèvement Forfaitaire Unique) taxe revenus capital à 30% au lieu du barème progressif (jusqu'à 62,2%). Objectif affiché : encourager investissement productif. Réalité : optimisation fiscale facilitée, revenus capital moins taxés que travail, pas de preuve d'investissement productif accru. Cadeau fiscal déguisé.

### Sources
- **Type** : ◉ Analyses économistes (IPP, Piketty)
- **Taux** : 30% (capital) vs jusqu'à 62% (travail)
- **Manque à gagner** : 1-2 Md€/an

### Informations complémentaires
- Flat tax : 30% revenus capital
- Avant : barème progressif (jusqu'à 62,2%)
- Bénéficiaires : épargnants riches
- Investissement productif : non prouvé
- Usage réel : optimisation fiscale
- Inégalité : capital < travail

## Mensonge #120

### Description
**Déclaration** : "Réforme fiscale pour classe moyenne"  
**Date** : 2017  
**Réalité** : 50% gains fiscaux vont aux 2% les plus riches

### Type de mensonge
**Λ (FRAMING)** - Présenter comme "classe moyenne" ce qui bénéficie aux très riches

### Contexte
La réforme fiscale (ISF, flat tax, impôt sociétés) est présentée "pour classe moyenne et compétitivité". L'IPP analyse : 50% des gains fiscaux bénéficient aux 2% les plus riches, 10% gagnants gagnent 5 500€/an, 90% inférieurs gagnent 150€/an. C'est une réforme pour très riches déguisée en réforme classe moyenne.

### Sources
- **Type** : ◈ Étude IPP (Institut Politiques Publiques)
- **Répartition** : 50% gains → 2% plus riches
- **Lien** : https://www.ipp.eu

### Informations complémentaires
- Réforme : ISF + flat tax + IS
- Présentation : "classe moyenne"
- Réalité IPP : 50% gains → 2% plus riches
- 10% plus riches : +5 500€/an
- 90% inférieurs : +150€/an
- "Classe moyenne" : mensonge, c'est pour très riches
