## II. NOUVEAUX PATTERNS DÉCOUVERTS (Batch Final 34→2)

### 1. **Séquence "On nous vend X / La réalité Y"** (Tweets 10, 9, 20, 23)

**Structure:**
```
On nous vend [narratif officiel dramatisé].
La réalité : [faits précis, nuancés, chiffrés].

──────────────────────────────
I - [Section titre]
[Contenu factualisé]

──────────────────────────────
CONCLUSION
[Verdict synthétique]
👉 [Formule finale percutante]
```

**Exemples:**
- 10.md: "On nous vend un scénario écrit d'avance : peuple prêt à tout bloquer [...] La réalité est plus triviale - et plus inquiétante."
- 9.md: "Une carte participative circule [...] Mais regardons sous la surface"

**Usage:** Déconstruire narratifs alarmistes ou simplificateurs, recentrer sur faits vérifiables.

## V. ANTI-PATTERNS DOCUMENTÉS

### 1. **Intro Méthodologique Académique** ❌
```
❌ "Cet article analyse [sujet] en utilisant une approche [méthodo]..."
✅ "On nous vend [narratif]. La réalité : [fait choc]."
```

### 2. **Numérotation H2/H3 Markdown dans Tweets Courts** ❌
```
❌ ## I. Section 1
✅ **I. Section 1** (gras inline)
OU
✅ ──────────────────────────────
    I - SECTION 1
```

**Exception:** Type D très long peut utiliser H2 markdown (ex: 93.md Hidalgo), mais JAMAIS Type E court.

### 3. **Sycophancy / Flagornerie** ❌
```
❌ "Vous avez raison, c'est un excellent point..."
❌ "Je suis d'accord avec votre analyse brillante..."
✅ "Les faits montrent que [X]."
✅ "Accuser sans preuves relève du [Y]."
```

### 4. **Vague / Approximatif** ❌
```
❌ "Beaucoup de Français sont mécontents"
✅ "63 % soutiennent le mouvement (Harris 20-21/08)"

❌ "La dette est énorme"
✅ "Dette 3 345 Md€, charge d'intérêts 59 Md€/an"
```

- **Type A/C/E:** JAMAIS H2, toujours gras inline

- Améliore lisibilité si ≥8 sections
