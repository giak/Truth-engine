## 🆕 Nouveaux Patterns Découverts

### 1. **TL;DR en Fin de Tweet Long** (76 Lombard)

**Pattern:**
```
[...tweet long développement...]

**TL;DR (sans fard)** :
– **Croissance** : **+0,8 %** officiel, **~+0,2–0,4 %** réel
– **Chômage** : **7,5 %** officiel, **12–15 %** élargi
– **Dette** : **113,9 % du PIB**
[...liste facts ultra-précis...]

🎯 **Objectif réel** : [formule finale]
```

**Usage:** Type D (note stratégique très longue)
**Avantage:** Synthèse bullet-points pour lecteurs pressés
**À AJOUTER au prompt:** Section optionnelle "TL;DR" pour Type D

---

### 2. **Trucage Narratif Numéroté** (65 sondages)

**Pattern:**
```
⚠️ **Trucage narratif n°1** : [mécanisme manipulation précis]
⚠️ **Trucage narratif n°2** : [autre mécanisme]
⚠️ **Trucage narratif n°3** : [etc.]
```

**Usage:** Déconstruction méthodologique Type C
**Similaire à:** "Contre-arguments démontés" mais focus sur manipulation narrative
**À AJOUTER au prompt:** Variante "Trucages narratifs" (déconstruction sondages/stats)

---

### 3. **Matrice Contradictions Structurée** (66 émojis)

**Pattern:**
```
**IV — Matrice des contradictions (6D)**

1. **Symbole vs acte** : [contradiction]
2. **Universalisme vs souveraineté** : [contradiction]
3. **Compassion vs capacité** : [contradiction]
4. **Écologie vs consommation** : [contradiction]
5. **Droits de l'homme vs realpolitik** : [contradiction]
6. **Consensus visuel vs pluralité réelle** : [contradiction]
```

**Usage:** Type D (analyse systémique contradictions)
**Structure:** Numérotation + **Concept A vs Concept B** : explication
**À AJOUTER au prompt:** Section optionnelle "Matrice contradictions" (6-10 dimensions)

### 6. **Comparaisons Chiffrées Précises** (65, 76, 69)

**Pattern ultra-précis:**

```
— **Garanties fortes** : **75 %** ✔️
— **France dans coalition** : **47 % / 36 % / 17 %** 🟰 **pluralité, pas majorité**
— **Troupes occidentales** : **47 % oui**, **53 % non** ❌
```

**Caractéristiques:**
- Émojis validation ✔️❌🟰
- Pourcentages exacts avec décimales si nécessaire
- Interprétation immédiate après chiffre
- Format tabulaire visuel

**À RENFORCER dans prompt:** Section "Facts ultra-précis" avec exemples comparaisons

### 7. **Sections Numérotées I-XII (Type D long)** (66)

**Structure découverte:**
```
**I — Titre section**
[Contenu]

**II — Titre section**
[Contenu]

[...]

**XII — Titre final**
[Contenu]
```

**Usage:** Type D très long (>3000 mots)
**Différence vs ## I.:** Pas de markdown H2, juste **gras numérotation romaine**
**À CLARIFIER dans prompt:** Type D peut utiliser **I.** OU ## I. (selon longueur)

```
— [Point 3] : **analyse**
```

**Usage:** Fin de Type D (résumé opérationnel)
**À AJOUTER au prompt:** Section optionnelle "Kit pratique" ou "Ligne claire"

## ⚠️ Anti-Patterns Confirmés

**Hashtags:**
- ✅ Acceptable UNIQUEMENT Type E très court (<400 mots)
- ❌ À éviter Types A/B/C/D
- Limite: <5 hashtags max
- Placement: toujours en fin

**Numérotation:**
- ✅ **I. II. III.** (gras) pour Type D très long (>3000 mots)
- ✅ ## I. ## II. (markdown) pour Type D moyen avec séparateurs
- ❌ "1. 2. 3." plat JAMAIS
