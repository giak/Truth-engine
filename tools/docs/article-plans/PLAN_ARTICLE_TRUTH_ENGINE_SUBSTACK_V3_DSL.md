**NOVEAU - ARCHITECTURE SYMBIONTIQUE LLM-DSL**:

**4.1 MÉCANISMES DE CONCEPTION DU DSL**:

**Inspiration Cognitive-Native**:

- Le DSL émerge des patterns de pensée du LLM, pas d'une conception externe
- Adaptation dynamique aux strengths/weaknesses du modèle de langage
- Optimisation pour le reasoning chain spécifique aux investigations
- Évolution continue basée sur les succès/échecs d'interprétation

**Principes Architecturaux Guiding**:

```
1. COGNITIVE ECONOMY: Maximum meaning, minimum tokens
2. SYMBOLIC PRECISION: Unambiguous notation for complex concepts
3. LLM-NATIVE: Aligned with transformer attention mechanisms
4. HIERARCHICAL NESTING: Multi-level abstraction support
5. PATTERN RECOGNITION: Optimized for detecting manipulation structures
```

**4.2 MÉTHODES D'INGÉNIERIE LINGUISTIQUE**:

**Symbolic Compression Language (SCL)**:

```
Avant (15 tokens): "ICEBERG pattern detected with factor 2.33 indicating moderate manipulation"
Après (3 tokens): "Ξ+(2.33)"
Gain: 80% compression, même précision conceptuelle
```

**Macro Expansion System**:

```
@PAT[ICEBERG] → [Population_Hidden + Shadow_Zones + Method_Switching + Context_Erasure]
@TRIGGER[TREE_LAUNCH] → [Complexity ≥ 9.0] + [Multi_perspective_required] + [APEX_protocol]

```

**Context-Aware Interpretation**:

- Le DSL s'adapte au contexte d'investigation en temps réel
- Résolution automatique d'ambiguïtés basée sur le domain
- Expansion intelligente selon les besoins de la requête

**4.3 PARADIGMES COGNITIFS INTÉGRÉS**:

**Hierarchical Abstraction Levels**:

```
L0: Raw Data → "7.4% unemployment rate"
```

- THESIS/ANTITHESIS/SYNTHESIS directement intégrées dans la syntaxe
- Opposition markers: ⟐⟐̅ pour perspectives contradictoires

**Contenu à développer**:

- **NOVEAU**: Normalisation des échelles (tout en 0-10 ou explication des différences)
- **IVF (Indice de Véracité Factuelle)**: 0-10, 4 composantes (V,C,S,T)
- **ISN (Indice de Suspicion Narrative)**: 0-10, pondération contextuelle
- **IVS (Indice de Vérité Systémique)**: 0-10, synthèse pondérée × Confidence
- **EDI (Epistemic Diversity Index)**: 0-10 (converti depuis 0-1), 6 dimensions
- **Targets par complexité**: SIMPLE≥3.0, MEDIUM≥5.0, COMPLEX≥7.0, APEX≥8.0

**NOVEAU - FORMULES NORMALISÉES**:

**IVF = avg(V,C,S,T) × (1-uncertainty) [0-10]**:

- V (Veracity): 10 - (contradictions×2) - (unsupported×0.5) - (Σ_manipulation×1.5)
- C (Context): 5 + history + cui_bono - omissions
- S (Sources): source_count×0.8 + EDI×3 + primary_evidence×2
- T (Transparency): method + citations_ratio×5 + formulas×2

**EDI = [(geo_diversity×0.25) + (lang_diversity×0.20) + (strat_diversity×0.20) + (ownership_diversity×0.15) + (perspective_diversity×0.15) + (temporal_diversity×0.05)] × 10**
