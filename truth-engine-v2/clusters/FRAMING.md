Parent symbol: Λ (LAMBDA/FRAMING)
- Score range: 0–10
- Score ≥5: cluster activates
- Score ≥7: full narrative architecture mapping
- Score ≥9: total narrative control flag

## Triggers
- Λ ≥5: Load all concepts, begin pattern detection
- Λ ≥7: Activate ALL concepts, map entire narrative architecture, identify frame controllers, search suppressed alternatives, break every false dilemma
- Λ ≥9: TOTAL NARRATIVE CONTROL — demand alternative perspectives, search censored/banned content, map propaganda architecture

## Queries

### FALSE_DILEMMA
**Pattern**: Only two choices presented
**Detection**: "Either... or" framing; third option never mentioned; nuance eliminated
**Queries**: "alternatives to", "third way", "other options"

### OVERTON_WINDOW
**Pattern**: Acceptable opinion boundaries
**Detection**: "Extreme" label for common sense; "Moderate" label for actual extreme; discourse boundaries enforced
**Queries**: "public opinion range", "acceptable discourse", "mainstream vs"

### ANCHORING_BIAS
**Pattern**: First number sets reference
**Detection**: Arbitrary starting point given; all comparisons to that anchor; real baseline hidden
**Queries**: "compared to what", "baseline year", "reference point"

### METAPHOR_CONTROL
**Pattern**: Metaphors shape thinking
**Detection**: War metaphors for non-war; market metaphors for non-market; natural metaphors for artificial
**Queries**: "actually means", "literal definition", "not metaphor"

### QUESTION_FRAMING
**Pattern**: Questions determine answers
**Detection**: Leading questions dominant; assumptions built into query; real question never asked
**Queries**: "underlying assumption", "real question", "presupposition"

### DEFINITION_GAMING
**Pattern**: Win by defining terms
**Detection**: Key terms redefined quietly; technical vs common definition; goalpost moving via definitions
**Queries**: "definition of", "what counts as", "technically means"

### BOUNDARY_DRAWING
**Pattern**: In-group/out-group creation
**Detection**: "Us vs them" framing; arbitrary boundary lines; unity-preventing divisions
**Queries**: "who is included", "boundary between", "division created"

### EUPHEMISM_TREADMILL
**Pattern**: Language laundering reality
**Detection**: Harsh reality softened; "collateral damage" not "dead civilians"; "restructuring" not "mass layoffs"
**Queries**: "really means", "plain language", "actual impact"

### NARRATIVE_MONOPOLY
**Pattern**: Single story dominates
**Detection**: One narrative everywhere; counter-narratives absent; story coordination visible
**Queries**: "alternative narrative", "different perspective", "suppressed story"

### AGENDA_SETTING
**Pattern**: Control what's discussed
**Detection**: Some topics prominent; others mysteriously absent; timing of topic introduction
**Queries**: "why now", "what's not discussed", "agenda behind"

## Deep Dive Protocol
When score ≥7, execute all 5 hermeneutic expansions:

1. **FRAME BREAKING** — "What's outside this frame?" → Search excluded alternatives, reveal hidden options
2. **ASSUMPTION HUNTING** — "What assumptions are built in?" → Surface presuppositions, challenge unquestioned premises
3. **METAPHOR ARCHAEOLOGY** — "What metaphors control thinking?" → Identify dominant metaphors and shaping power
4. **BOUNDARY ANALYSIS** — "Who drew these boundaries and why?" → Trace boundary creation, cui bono from divisions
5. **NARRATIVE COUNTER** — "What's the suppressed story?" → Find alternative narratives, amplify silenced voices

## Output Format
- Concept matches with detection evidence
- Query boost results per pattern
- Frame controller identification
- Suppressed alternatives list
- Trigger escalation level (5/7/9)
