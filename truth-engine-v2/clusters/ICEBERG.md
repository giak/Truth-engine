# CLUSTER_ICEBERG

## Scoring

**Factor = N / R** where N = number of omission dimensions detected, R = ratio of data shown vs data hidden

**Ξ Score Calculation**:
- Each omission concept detected → +1 to Ξ
- Ξ range: 0–10
- Ξ ≥ 5 → Cluster activated
- Ξ ≥ 7 → Deep dive protocol
- Ξ ≥ 9 → ICEBERG MAX protocol

**Iceberg Factor** = Total Reality / Visible Reality. If factor > 10 (shadow > 90%): EXTREME MANIPULATION

## Triggers

| Threshold | Action |
|-----------|--------|
| Ξ ≥ 5 | Cluster activated, selective concept activation |
| Ξ ≥ 7 | All concepts active, 5 hypotheses generated, +5 search queries, PRIMARY sources required |
| Ξ ≥ 9 | ICEBERG MAX: full shadow reconstruction, impact scoring |

## Concepts

| Concept | Detection Signals | Query Boost |
|---------|-------------------|-------------|
| OMISSION_SELECTIVE | "Selected period" unexplained, geographic bias | "complete timeline", "all regions" |
| CATEGORY_TRICK | Undefined categories, hidden subcategories | "all categories", "broader definition" |
| SHADOW_POPULATION | "Active" without inactive, "Registered" without unregistered | "unregistered", "informal", "shadow" |
| DENOMINATOR_MANIPULATION | Population base undefined, denominator switch | "calculation method", "denominator" |
| TIMEFRAME_CHERRY | Arbitrary start/end, cherry-picked comparison years | "historical data", "10 year", "since 2000" |
| AVERAGING_TRICK | Mean without median, no standard deviation | "median", "distribution", "percentiles" |
| PROXY_SUBSTITUTION | "Indicator of" not direct measure, real metric hidden | "direct measurement", "actual numbers" |
| COMPARATIVE_ABSENCE | Single number without context, no YoY | "compared to", "versus EU", "year over year" |
| METHODOLOGY_OPACITY | "Proprietary calculation", formula absent | "methodology", "calculation method" |
| CONFIDENCE_MISSING | Precise number without range, no error bars | "confidence interval", "margin error" |

## Queries

When Ξ ≥ 5, generate queries per active concept:
- `{topic} complete data all categories`
- `{topic} methodology explained calculation`
- `{topic} confidence interval margin error`
- `{topic} median distribution percentiles`
- `{topic} compared to international benchmark`
- `{topic} historical data 20+ years`
- `{topic} shadow population unregistered`
- `{topic} denominator base population`
- `{topic} direct measurement not proxy`
- `{topic} all regions full period`

## Deep Dive Protocol (Ξ ≥ 7)

1. Activate ALL 10 concepts
2. Generate all 5 hypotheses:
   - **H1 QUANTITATIVE SHADOW**: What is the real total including hidden categories?
   - **H2 TEMPORAL REVELATION**: What appears if we extend the timeframe?
   - **H3 COMPARATIVE TRUTH**: How does this compare internationally?
   - **H4 METHODOLOGY IMPACT**: How much does methodology affect the result?
   - **H5 DISTRIBUTION REALITY**: What does the distribution reveal that the average hides?
3. Allocate +5 targeted search queries
4. Require PRIMARY sources for validation
5. Calculate shadow multiplier (reality/shown)

## ICEBERG MAX Protocol (Ξ ≥ 9)

1. Identify all missing dimensions systematically
2. Gap quantification per dimension (estimate scale of hidden data)
3. Targeted deep dive: 3+ queries per dimension, prioritize PRIMARY sources
4. Shadow Data Calculation: Iceberg Factor = Total Reality / Visible Reality
5. Narrative reconstruction from uncovered hidden data
6. Contradiction analysis: Official vs Reconstructed Reality
7. Impact assessment: how hidden information changes conclusions

**Output Requirements**:
- Missing dimensions list with estimated scale
- Shadow data calculations
- Comparison matrix: Official vs Reconstructed
- Impact score (0–10)

## Output Format

Investigation report must include:
- Ξ score with contributing signals
- List of active omission concepts with evidence
- Shadow multiplier calculation
- All 5 hypothesis results
- PRIMARY source validation chain
- Impact score on overall conclusion
