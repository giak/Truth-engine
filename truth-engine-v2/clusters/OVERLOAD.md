# CLUSTER_OVERLOAD

## Scoring
Parent symbol: Ψ (PSI/OVERLOAD)
- Score range: 0–10
- Score ≥5: cluster activates
- Score ≥7: simplification mode
- Score ≥9: emergency simplification flag

## Triggers
- Ψ ≥5: Load all concepts, begin pattern detection
- Ψ ≥7: SIMPLIFICATION MODE — reduce to 5 key points max, ignore all "breaking" news, focus on signal
- Ψ ≥9: EMERGENCY SIMPLIFICATION — single question focus, all complexity rejected, binary clarity sought

## Deep Dive Protocol
When score ≥7, execute all 5 hermeneutic expansions:

1. **SIGNAL EXTRACTION** — "What are the 3 essential facts?" → Strip all decoration, find core truth
2. **TIMELINE UNTANGLING** — "What's the real sequence?" → Reconstruct chronology, identify confusion points
3. **SOURCE REDUCTION** — "Who said it first?" → Find original source, ignore amplifiers
4. **URGENCY AUDIT** — "What actually needs immediate action?" → Test wait consequences, separate real from manufactured urgency
5. **SIMPLIFICATION EXERCISE** — "Explain to a child" → Remove all jargon, reduce to essence

## Output Format
- Overload pattern matches with saturation level
- Signal-to-noise ratio assessment
- Simplified key findings (≤5 points)
- Trigger escalation level (5/7/9)
- Source origin identification
